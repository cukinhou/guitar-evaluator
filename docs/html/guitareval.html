guitareval.guitar_evaluator module¶

This script implements the class GuitarEvaluator for guitar onset and pitch analysis. The onset detection system uses Sebastian Bock’s algorithm, based on pre-trained Convolutional Neural Networks, implemented in Madmom. Pitch estimation is addressed using Justin Salomon’s Melodia algorithm implemented in Essentia library.

author: Javier Nistal
